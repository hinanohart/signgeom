# plugins/egan-applets — License pending

This subdirectory is **reserved** for a future plugin that will reimplement
classic interactive physics applets (light cones in (4,0) Orthogonal geometry,
Dichronauts (2,2) raymarchers, Penrose–Terrell rotation demos, etc.) using
**public-domain mathematics only**.

**Status as of 2026-05-17 (v0.1.0):** empty. No source files. No assets.

## Rules for any future code under this directory

1. **No inspection** of `gregegan.net` applet source (HTML/JS/CSS). The 35
   applets hosted there are © Greg Egan, all rights reserved.
2. **Source the math from public domain only**: Klein quartic (1879),
   24-cell (Schläfli), Penrose–Terrell (1959), Born rigidity (1909),
   Mermin (1990), Unruh (1976), Schwarzschild (1916), etc.
3. **One-line attribution allowed**: `Inspired by Greg Egan's <title>.`
   No quotation of Egan's prose, no trademark use, no logo.
4. **Independent re-derivation**: every formula must be reproducible from a
   citable public source (textbook, arXiv paper, Wikipedia with reference).

## v0.2+ decision point

A separate review (R14 round) will decide whether to ship the applet
reimplementation pack in v0.2 or to keep this directory reserved indefinitely.

See `../../docs/book/src/license-strategy.md` for the full reasoning.
